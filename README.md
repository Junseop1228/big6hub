## Quick Start
```bash
npm install
cp .env.example .env
npm run seed
npm start
```

## Tech Stack
- Backend: Node.js + Express
- Database: SQLite
- Frontend: Vanilla HTML / CSS / JS
- Auth: JWT + bcrypt
- Docs: Swagger / OpenAPI
- CI: GitHub Actions
